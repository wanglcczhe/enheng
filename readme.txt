0、首页顶部可登录--注册--注销
	注册：首页点击注册后，跳转到注册页面，账号，密码随意；有注册成功的提示；可以通过注册按钮下面的登录链接跳转到登录页面，也可以通过返回首页点击登录，进入登录页面。
	登录：有登录失败的提示，账号密码不对或者没有注册都会触发；登录成功后，首页顶部左上角会显示注册的用户名；
